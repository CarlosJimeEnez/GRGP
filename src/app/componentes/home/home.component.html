Kart identification

Our kart identification system uses UWB (Ultra Wideband) technology to provide accurate, real-time location of each kart on the track.

Improved safety through speed control

Our speed monitoring tool, based on an advanced radio protocol, offers a revolutionary solution to improve road safety. Designed to reduce accidents

Crash Detection and Real-Time

Our system uses accelerometers to detect sudden impacts. When a crash is detected, it sends alerts and records the location using GPS. This ensures quick response and improved safety.
